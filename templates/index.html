    Quick Instructions

    1. Install deps: pip install -r requirements.txt
    2. Train: python train.py (writes model_best.pth)
    3. Run UI: python app.py then open http://localhost:8000
    4. Draw + Predict: view digit, confidence, activations
    5. After retrain: click “Reload model” to pick up new weights
